# Яндекс.Вебмастер: проверка и индексация

Проверку и настройку нужно выполнить вручную в [Яндекс.Вебмастер](https://webmaster.yandex.ru/).

## 1. Добавить сайт (если ещё не добавлен)

- Зайти в [webmaster.yandex.ru](https://webmaster.yandex.ru/).
- Добавить сайт: `https://www.setki21.ru`.
- Подтвердить права (файл в корне, мета-тег или DNS — как предложит сервис).

## 2. Диагностика сайта

- Раздел **«Диагностика»** (или **«Проверка»**).
- Дождаться проверки по ~40 параметрам.
- Исправить все **критические** замечания:
  - отсутствующие мета-теги (title, description);
  - ошибки 4xx/5xx по важным URL;
  - проблемы с оригинальностью контента (дубли);
  - недоступные целевые страницы.

## 3. Sitemap

- Раздел **«Индексирование»** → **«Файлы Sitemap»**.
- Указать: `https://www.setki21.ru/sitemap.xml`.
- Дождаться обработки. В sitemap сейчас: главная, категории, ремонт, privacy, karta-sajta (8 URL).

## 4. Проверка индекса

- Раздел **«Индексирование»** → **«Страницы в поиске»** (или **«Индекс»**).
- Убедиться, что в индексе есть все нужные страницы:
  - `/`
  - `/antimoshka`, `/antikoshka`, `/antipyl`, `/vstavnye`, `/remont`
  - `/privacy`, `/karta-sajta`
- Если каких-то страниц нет — запросить переобход в разделе переобхода URL (если доступно).

## 5. Регион и контакты

- В настройках сайта указать **регион** (Чебоксары / Чувашская Республика).
- Проверить, что контакты и адреса соответствуют данным на сайте и в карточке Яндекс.Бизнес.

## После изменений на сайте

- После правок sitemap или важных страниц можно снова отправить sitemap или запросить переобход изменённых URL в Вебмастере.
