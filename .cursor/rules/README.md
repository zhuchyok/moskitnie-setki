# Правила Cursor для проекта jitro

## Автоматически загружаемые правила

- `essential.mdc` - Критичные правила проекта (всегда активны)

## Правила по запросу

Загружайте через `@filename` когда нужно:

- `@project_reference.mdc` - Справочник компонентов проекта
- `@cursor_agent_coordinator.mdc` - Координатор агентов
- `@victoria-tunnel.mdc` - Туннель к Виктории (все сообщения уходят в VictoriaATRA)
- `@global-rules.mdc` - Глобальные правила workflow
- `@python_engineer.mdc` - Роль Python инженера
- `@ml_engineer.mdc` - Роль ML инженера
- И другие роли по необходимости

## Как использовать

Вместо того чтобы все правила загружались автоматически, теперь вы можете:

1. **Для обычных задач:** Используйте только `essential.mdc` (загружается автоматически)
2. **Для сложных задач:** Добавьте нужные правила через `@filename`
3. **Для работы с компонентами:** Используйте `@project_reference.mdc`

Это значительно снижает расход токенов!
